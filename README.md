# Project Desc.
这是一个关于英国交通的项目。项目的结构上采用了前后端分离。
其中：
- traffic-fe: frontend
  是前端部分，使用vue

- traffic-be: backend
  是后端部分，使用flask

## Quick start
启动后端
- cd traffic-be
- pip3 install -r requirements.txt
- python3 app.py
run server which is listening on port 20000.

启动前端
- cd traffic-fe
- npm install
- npm run dev
run pages on port 3000/3001.
